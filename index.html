Razzles Run

An endless runner game assigned as a first project.

About

This was our first college project, where we were tasked with creating a solo endless runner. Each aspect of the game had to be developed individually.

Process

Art/Visuals: I used Photoshop for backgrounds, buttons, and static images, and Piskel (a simpler online alternative to Aseprite) for animations.

Sound/Music: We used BandLab to create background music, which suited my limited music-making skills. For sound effects, I either recorded my own or sourced them online, editing them with Audacity.

Code: At the start of this project, I had no prior programming experience. Initially challenging, programming became enjoyable, and I steadily improved my skills as the project progressed.
